# API Specification (Overview)

This document will contain the REST API endpoints for authentication, users, messages, groups, and media. Start with:

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/users/:id`
- `POST /api/messages` (send)
- `GET /api/messages?conversationId=...`

Detailed OpenAPI-style specs will be added per phase.
